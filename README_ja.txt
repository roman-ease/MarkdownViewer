Markdown Viewer v1.1

ローカル環境向けに構築された、高速・軽量なMarkdownビューア兼エディタです。
Windows 10 / 11 の標準コンポーネント（WebView2）を利用してシームレスに動作します。

■ Ver 1.1 の新機能
- **複数タブ対応**: 複数のファイルを同時に開き、タブで切り替えて編集できます。
- **Mermaid対応**: グラフやフローチャートを Markdown 内で描画できます。
- **Obsidian型リンク & 画像貼り付け**: `![[画像]]` 形式のサポートに加え、クリップボードからの画像貼り付け（自動保存＆リンク挿入）に対応しました。


■ 使い方
1. ダウンロードした ZIP ファイルを、お好きなフォルダに「すべて展開（解凍）」してください。
2. 展開されたフォルダ内にある `MarkdownViewer-win_x64.exe` をダブルクリックするだけで起動します。

■ .md ファイルをダブルクリックで開く設定
Windowsで .md ファイルをダブルクリックした際に常にこのアプリで開きたい場合は、以下の設定を行ってください。
1. 任意の `.md` ファイルを右クリックします。
2. 「プログラムから開く」 > 「別のプログラムを選択」 をクリックします。
3. リストの一番下にある「その他のアプリ」または「PCで別のアプリを探す」をクリックします。
4. 本アプリの `MarkdownViewer-win_x64.exe` を選択して「開く」を押します。
5. 「常にこのアプリを使って .md ファイルを開く」などのチェックボックスをオンにしてください。

■ 制約事項（D&Dに関する注意）
Chromium(WebView2)のセキュリティ制限により、エクスプローラなどから画面上にファイルをドラッグ＆ドロップ（D&D）した際、元のファイルの正しい位置（パス）を取得できません。
そのため、ファイルの中身だけを読み込み、アプリ上では**「パスを持たない新規ファイル」**として扱われます。直接の上書き保存はできず「名前を付けて保存」になる点、および画像を相対パス指定（`./image.png` 等）している場合に正しく表示されない点にご注意ください。

■ 同梱ファイル一式（同一フォルダに置いてお使いください）
- MarkdownViewer-win_x64.exe （本体）
- resources.neu （画面等のリソースデータ）
