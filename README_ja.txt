Markdown Viewer v1.5.2

ローカル環境向けに構築された、高速・軽量な Markdown ビューア兼エディタです。
Windows 10 / 11 の標準コンポーネント（WebView2）を利用してシームレスに動作します。

■ 主な特徴 (v1.5.2)
- **軽量な配布サイズ**: 実行バイナリとリソースを含めて約2.3MBのコンパクトな設計。
- **高速な起動・操作**: Neutralinojs フレームワークを採用し、リソース消費を最小限に。
- **マルチタブ対応**: 複数のファイルを同時に開き、タブで切り替えて編集・閲覧が可能。
- **Excel/TSV 連携**: Excel のセルをコピーして貼り付けるだけで Markdown テーブルに自動変換。
- **同期スクロール**: エディタとプレビューのスクロール位置を連動（ツールバーでON/OFF切替）。
- **マルチテーマ**: ダーク / ライト / セピアの3モードから最適な視覚環境を選択可能。
- **DOMPurify によるセキュリティ**: Markdown 解析後の HTML をサニタイズし、XSS 攻撃のリスクを低減。
- **Mermaid/数式・コード対応**: グラフ描画（Mermaid）、数式、シンタックスハイライト（Highlight.js）をサポート。
- **画像貼り付け支援**: クリップボードからの画像貼り付け、自動保存、リンク挿入に対応。

■ 使い方
1. ダウンロードした ZIP ファイルを、お好きなフォルダに「すべて展開（解凍）」してください。
2. 展開されたフォルダ内にある `MarkdownViewer.exe` をダブルクリックして起動します。
   ※ 初回起動時に「Windows によって PC が保護されました」と表示される場合は、「詳細情報」→「実行」を押してください。

■ .md ファイルをダブルクリックで開く設定
1. 任意の `.md` ファイルを右クリックし、「プログラムから開く」 > 「別のプログラムを選択」 を選択。
2. 「PCで別のアプリを探す」から本ソフトの `MarkdownViewer.exe` を指定。
3. 「常にこのアプリを使って .md ファイルを開く」にチェックを入れて完了です。
