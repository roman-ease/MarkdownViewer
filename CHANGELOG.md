# Changelog

## [1.5.1] - 2026-03-19

### Changed

- **UI 構造の整理**: 「テーブル」ボタンを新設の「ツール」メニュー内に統合。今後の機能拡張に対応。
- **テーブル作成の高度化**: 挿入時にダイアログで「行数x列数」を指定可能に。

### Fixed

- **テーマ切替不具合**: デザインを選択しても反映されない問題を修正（CSS 変数の適用範囲をルートレベルへ拡張）。

## [1.5.0] - 2026-03-19

### Added

- **テーブル編集支援**: ツールバーに「テーブル挿入」ボタンを追加し、3x2 の Markdown テーブル雛形を即座に挿入可能に。
- **Excel/TSV 貼り付け変換**: Excel やスプレッドシートのセルをコピーして貼り付ける際、自動的に Markdown テーブル形式へ変換する機能を実装。画像貼り付けとの競合も解消。
- **マルチテーマ対応**: 「ダーク（標準）」に加え、目に優しい「ライト」と「セピア」のテーマ選択機能を追加。設定はブラウザの `localStorage` に保存され、再起動後も維持。
- **同期スクロール**: エディタとプレビューのスクロール位置をリアルタイムに同期。ツールバーの「同期ON/OFF」ボタンで切り替え可能。

## [1.4.0] - 2026-03-18

### Added

- **セキュリティ強化 (XSS対策)**: `DOMPurify` を導入し、Markdown 描画時のサニタイズを徹底。悪意あるコードの実行を防止。
- **コード構造の刷新 (モジュール分割)**: 肥大化した `app.js` を 7 つの独立したモジュール (`state`, `ui`, `preview`, `file`, `tabs`, `ipc`, `editor`) に分割。保守性と視認性を向上。

### Changed

- **リポジトリの専門家対応**: 不要なサンプルファイル (`main.js`) や直置き ZIP、開発環境由来のディレクトリ (`.tmp`, `.vs`) を削除。
- **.gitignore の整備**: 開発環境依存ファイルやビルド成果物を適切に除外。

---

## [1.3.0] - 2026-03-18

### Added

- IPC（プロセス間通信）の高速化。ストレージ監視間隔を `200ms` に短縮し、他ファイルからの遷移を高速化。
- 履歴機能の自動移行処理。旧バージョンのタイポキー (`merkdown`) から正しいキー (`markdown`) へ自動引き継ぎ。

### Fixed

- セカンダリインスタンス起動時の終了待機時間を最適化し、ウィンドウ消去速度を向上。
- 不要なデバッグ用コード、重複コメント、未使用の `try-catch` などを排除。

---

## [1.2.0] - 2026-03-18

### Added

- シングルインスタンス化（二重起動防止）機能を正式実装。
- IPC（プロセス間通信）の仕組みを `Neutralino.storage` ベースの設計に刷新。

### Fixed

- ストレージ操作権限（`storage.*`）の欠落により、特定環境でタブ追加が失敗していた不具合を修正。
- 一時フォルダのアクセス権に依存しない安定したメッセージングを実現。
- セカンダリインスタンス起動時のウィンドウ明滅現象を抑制。

---
