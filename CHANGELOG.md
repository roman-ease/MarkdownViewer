# Changelog

## [1.5.5] - 2026-03-19

### Fixed

- **セキュリティ権限の最小化 (Least Privilege)**: `nativeAllowList` のワイルドカード（*）を廃止し、実際に使用している関数のみを許可するように制限。アタックサーフェスを削減し、安全性を向上。
- **未使用モジュールの削除**: `clipboard.*` および `events.*` への Native API アクセス権限を削除（標準 Web API で代用しているため）。

### Fixed

- **関連付け起動時の終了不具合修正**: エクスプローラー等でのダブルクリック（ファイル関連付け）から起動した際、ウィンドウが閉じなくなる不具合を修正。終了イベント発生時に直ちにファイル監視処理を停止させることで、リソースのロックを回避。
- **診断機能の強化**: 起動時のカレントディレクトリ（CWD）や引数をログに出力する機能を追加。

### Fixed

- **終了処理の堅牢化**: ウィンドウの「✕」ボタンを押した際、特定の環境（権限不足等）でアプリが終了しない不具合を修正。例外保護を強化し、確実にプロセスが終了するように改善。
- **IPC通信の安定化**: ダブルクリック起動によるタブ追加が機能しない不具合に対処。ストレージアクセス失敗時でもアプリがハングアップしないようハサミ処理を強化。

### Fixed

- **UI制御の安定化**: ツールメニュー（開かない不具合）およびテーマ切替（反映されない不具合）のイベントリスナーを整理・統合し、レスポンスを改善。
- **テーマ定義の修正**: CSS 変数の適用範囲をルート要素（html）に拡張し、デザイン変更が即座に同期されるよう修正。

## [1.5.1] - 2026-03-19

### Changed

- **UI構造のリファクタリング**: 「テーブル」ボタンを新設の「ツール」ドロップダウンメニューへ統合。今後の機能拡張（ユーティリティ追加等）に対応可能な設計へ変更。
- **テーブル作成の高度化**: 挿入時にダイアログで「行数x列数」を指定可能にし、自由なサイズのテーブル雛形を作成可能に。

## [1.5.0] - 2026-03-19

### Added

- **Excel/TSV 連携**: Excel やスプレッドシートのセルをコピーして貼り付ける際、Markdown テーブル形式へ自動変換する機能を実装。
- **同期スクロール**: エディタとプレビューのスクロール位置をリアルタイムに同期。ツールバーの「同期ON/OFF」ボタンで切り替え可能。
- **マルチテーマ対応**: ダーク（標準）、ライト、セピアの3テーマを選択可能にし、設定を `localStorage` に保存。

## [1.4.0] - 2026-03-18

### Added

- **モダン・タブシステム**: 複数ファイルを同時に開き、タブで切り替えて編集・閲覧できる統合環境を構築。
- **セキュリティ強化 (XSS対策)**: `DOMPurify` を導入し、Markdown 解析後の HTML サニタイズを徹底。
- **コード構造の刷新**: 内部設計をモジュール方式 (`state`, `ui`, `editor` 等) にリファクタリングし、保守性を向上。

### Changed

- **リポジトリ清掃**: 不要なサンプルスクリプトや開発用の一時ディレクトリ、ビルド済み ZIP 等を削除し、公開用としての品質を確保。

---

## [1.3.0] - 2026-03-18

### Added

- IPC（プロセス間通信）の高速化。ストレージ監視間隔を `200ms` に短縮し、他ファイルからの遷移を高速化。
- 履歴機能の自動移行処理。旧バージョンのタイポキー (`merkdown`) から正しいキー (`markdown`) へ自動引き継ぎ。

### Fixed

- セカンダリインスタンス起動時の終了待機時間を最適化し、ウィンドウ消去速度を向上。
- 不要なデバッグ用コード、重複コメント、未使用の `try-catch` などを排除。

---

## [1.2.0] - 2026-03-18

### Added

- シングルインスタンス化（二重起動防止）機能を正式実装。
- IPC（プロセス間通信）の仕組みを `Neutralino.storage` ベースの設計に刷新。

### Fixed

- ストレージ操作権限（`storage.*`）の欠落により、特定環境でタブ追加が失敗していた不具合を修正。
- 一時フォルダのアクセス権に依存しない安定したメッセージングを実現。
- セカンダリインスタンス起動時のウィンドウ明滅現象を抑制。

---
